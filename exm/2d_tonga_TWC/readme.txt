# See documentation dNami/doc/ folder by compiling it. To do so, run:

make html 

while in dNami/doc/ and then navigate to dNami/doc/_build/html and open index.html with a browser to browse the documentation. The documentation for this case can be found in the 'Detailed examples' tab. 
 
Note: this case currently runs with a custom version of dnami_mpi.py. Please copy the file in the this folder to dNami/src/src_py.
